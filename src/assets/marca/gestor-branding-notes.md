# Gestor — ativos de marca

Arquivos:
- `gestor-logo.svg`: logo horizontal principal
- `gestor-logo-icon.svg`: versão compacta
- `favicon.svg`: favicon vetorial
- `favicon-32.png`: favicon raster 32x32
- `app-icon-192.png`: app icon 192x192
- `app-icon-512.png`: app icon 512x512

## Cores principais
- Primária: `#2563EB`
- Fundo padrão dark: `#020617`
- Surface: `#0F172A`
- Texto: `#F1F5F9`

## Tailwind
Use:
- `bg-slate-950` para fundo global
- `bg-slate-900` e `bg-slate-950/80` para superfícies
- `text-slate-100`, `text-slate-300`, `text-slate-400`
- `bg-blue-600 hover:bg-blue-500`
